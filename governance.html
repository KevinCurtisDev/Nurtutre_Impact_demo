Governance Masterclass Series

Masterclass 1

Foundations of Good Governance in Irish Nonprofits

Build a shared understanding of what good governance looks like in practice for community organisations.

Core Content

  • What governance is and is not
  • Roles vs responsibilities: board, chair, CEO/manager, volunteers
  • Overview of the Charities Governance Code
  • Legal structures in Ireland (CLG, Co-op, Unincorporated, Charity)
  • Fiduciary, strategic, and oversight duties
  • The governance–operations boundary

Tools & Takeaways

  • Governance role map
  • Board responsibilities checklist
  • Governance Code compliance starter template
Masterclass 2

The Charities Governance Code — Practical Compliance

Demystify the Code and support organisations to implement it meaningfully, not mechanically.

Core Content

  • The six principles explained in plain language
  • Core vs additional standards
  • Evidence requirements and common pitfalls
  • How to embed the Code into everyday practice
  • Annual compliance reporting

Tools & Takeaways

  • Sample compliance record form
  • Evidence bank template
  • Board self-assessment tool
